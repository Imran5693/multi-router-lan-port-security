# Multi-Router LAN with Wireless Integration and Port Security Simulation

This project simulates a **secure multi-segmented LAN** using **Cisco Packet Tracer**, including **wireless integration**, **port security**, and **rogue device blocking**.

## Features
- Multi-router LAN with three subnets
- RIP v2 dynamic routing configuration
- Wireless access points with WPA2 security
- Port security to block unauthorized devices
- Full IP addressing table and topology screenshots

## Files
- `multi-router.pkt` → Packet Tracer simulation
- `screenshots/` → Topology, IP tables, and rogue device demonstration
- `report/project_report.pdf` → Comprehensive project report
- `video_links.md` → YouTube demo

## How to Use
1. Open `multi-router.pkt` in **Cisco Packet Tracer**.
2. Observe LAN configuration, routing, wireless integration, and port security.
3. Check screenshots and PDF report for detailed explanation.
